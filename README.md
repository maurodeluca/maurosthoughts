# maurosthoughts.com

A minimalist, alternative personal blog that unfolds like a manifesto. Features:

- Terminal-style "thinking" line with blinking cursor
- Typed quotes that appear on scroll
- Scroll-lock micro-pauses to guide user attention
- Hidden navigation revealed at the end
- Editorial, red/black color palette with minimal design

## Future plans

- Convert content to Markdown for easier writing
- Upgrade to Astro for static site generation
- Publish long-form essays directly from markdown files

## Usage

1. Clone the repository
2. Open `index.html` in a browser
3. Or deploy to GitHub Pages / Netlify for live hosting
